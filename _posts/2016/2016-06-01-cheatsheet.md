## Linux

**wget download**   
```sh
wget -r --no-parent http://abc.tamu.edu/projects/tzivi/repository/revisions/2/raw/tzivi/
```

**GNU screen**   

```sh
screen   
C-p d       # detach current screen session   
screen -r   # resume last screen session   

C-p <p>"</p>       # list windows   
C-p A       # channge window name   
C-p 0       # open window #0    
C-p S       # split    
C-p C-p     # swap window    
```
